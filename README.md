# AI_Projekt
Detta projekt går ut på att en AI ska skapa en låt vilket den ska generera ett lyrics och en "sång" av.


## Code
### Create_RapSong 
Scriptet skapar ett lyrics till en egengenererad låttext som den sjunger/läser upp.
